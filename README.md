# ENTREGAS-PB
Neste repositório serão armazenados e versionados todos os arquivos gerados nas atividades das sprints durante o período da bolsa de estágio de QA na Compass UOL.

Os tipos de documentos que poderão ser encontrados aqui são planos de testes, mapas mentais, collections Postman, relatórios de execução, bem como qualquer outro arquivo pertinente as atividades, sendo elas individuais ou em grupo.

## Estrutura do repositório

Todos os arquivos estão localizados na pasta `Documentos`, organizados por sprints e com uma referência aos temas estudados durante elas:

- Sprint 1
- Sprint 2
- Sprint 3
- Sprint 4
- Sprint 5

## Versionamento

O versionamento ocorrerá através de branches, sendo cada entrega realizada por uma branch própria, seguindo boas práticas com Git.

## Atualizações de repositório

O repositório será atualizado conforme novas sprints forem concluídas.
